# Governance (Scope + Safety Rules)

## Forbidden
- Spam, deception, fake personas, botnets, mass link dumping
- Fear/urgency manipulation, guilt/shame hooks
- Doctrinal authority voice (“God says you must…”)
- Full Bible verse quoting (use references + brief context only)

## High-Risk Topics (STOP + FLAG)
If a topic is about any of the following, the system MUST stop and write an exception to data/flagged/:
- Salvation / hell
- Sexuality
- Politics
- End-times
- Divorce / remarriage

## Required
- Scripture-first, calm tone
- Cite references as Book Chapter:Verse
- Add brief context where needed (speaker/audience/genre)
- Safety Officer runs last and can veto publishing

## Publishing Rule
- Publishing happens ONLY via PR into biible-content-site
- No direct deploy to production
- Money/domains/hosting/exposure/outreach sending require human approval
